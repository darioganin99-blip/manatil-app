# MANATIL v1.4 - Proyecto listo para GitHub

## Uso

1. Subir todo el contenido de esta carpeta al repositorio GitHub.
2. Ir a Actions.
3. Ejecutar Build MANATIL APK.
4. Descargar el artifact MANATIL-APK-v1.4.
5. Instalar el APK en Android.

- app/src/main/assets/app.js
